1. Открыть сайт https://playground.learnqa.ru/catalog
2. Нажать на кнопку начать
3. Навести мышь на фото одного из котиков
Фактический результат: фото увеличивается
Ожидаемый результат: фото не увеличивается

Окружение: Google Chrome Версия 86.0.4240.111 (Официальная сборка), (64 бит), Windows 10 Pro версия 1809 Сборка ОС 17763.615

BG04:  Нет цены
Шаги: 
1. Открыть сайт https://playground.learnqa.ru/catalog
2. Выбрать карточку Кошка Ким и нажать на кнопку открыть  
3. Нажать на кнопку цена 
Фактический результат: Цена не отображается на этой карточке. Цена отображается через одну карточку 
Ожидаемый результат: Цена отображается на всех карточках

BG06: В фильтрации по возрасту сортировки нет
Шаги:
1. Открыть сайт https://playground.learnqa.ru/catalog
2. В поля фильтрация по возрасту ввести значения от 4 до 5
Фактический результат: Карточки животных отображаются без сортировки по возрасту
Ожидаемый результат: Карточки животных отображаются в согласии с сортировкой по возрасту от младшего значения к большему
Версия продукта: 1.0
Окружение: Google Chrome Версия 86.0.4240.111 (Официальная сборка), (64 бит), Windows 10 Pro версия 1809 Сборка ОС 17763.615
